# Юридическая информация

## Определения

**Пространство имён (Namespace)** - поддиректория, начинающаяся с `_`, указывающая на авторство содержимого.

**Код** - любые файлы исходного кода C#, Python, скомпилированные сборки, YML-файлы в `Resources` и связанные скрипты (включая поддиректорию `Tools`).

## Авторские права

Контрибьюторы сохраняют авторские права на свои работы, предоставляя
безотзывную лицензию под [AGPL-3.0-or-later](https://github.com/Shegare/SIS-14/blob/master/LICENSE_AGPL.txt) для проекта [Skill Issue Station (SIS-14)](https://github.com/Shegare/SIS-14).

Это означает:
- Контрибьюторы могут повторно использовать свой код в других проектах
- Контрибьюторы НЕ могут отозвать лицензию на уже внесённый код
- Изменение лицензии проекта требует согласия всех правообладателей

## Лицензии кода

Основная лицензия проекта - AGPL-3.0-or-later.

Код, созданный участниками SIS-14 - находится в пространствах имён `_SIS`.

Для кода других авторов необходимо соблюдать условия как AGPL-3.0-or-later, так и их исходной лицензии (см. таблицу атрибуции ниже).
Информация об авторских правах для кода под AGPL-3.0-or-later приведена в тексте лицензии AGPL-3.0-or-later.

### Модификации на основе MIT-кода

Исходный код из [Space Station 14](https://github.com/space-wizards/space-station-14) и других MIT-проектов может быть модифицирован и перелицензирован под AGPL-3.0-or-later.

Текущая лицензия таких файлов определяется по SPDX-заголовку. Например:

```
// SPDX-FileCopyrightText: [оригинальные правообладатели]
// SPDX-FileCopyrightText: 2025 Skill Issue Station contributors
//
// SPDX-License-Identifier: AGPL-3.0-or-later
```

Если в коде файлов отсутствуют SPDX-заголовки - файл остаётся под исходной лицензией.

## Особые требования к ассетам

Лицензии ассетов указаны в файлах `attributions.yml` и `meta.json` - соблюдайте их индивидуально.

## Устаревший код

До [коммита 50d45b5](https://github.com/Shegare/SIS-14/commit/50d45b5298a03542a9976526b7bc39fd664146e6) YML-файлы в директории `Resources` не имели явной лицензии.

Если вы не согласны с текущими условиями AGPL-3.0-or-later:
- Вы можете использовать YML-файлы из коммитов до 50d45b5 включительно
- Эти версии файлов доступны под лицензией MIT (как часть Space Station 14)
- Более поздние версии YML-файлов лицензированы согласно разделу "Лицензии кода" выше
